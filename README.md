## 整除  
**若a是b的因数,或b是a的倍数,则a整除b,记作a∣b**  
* 关于整除,有以下几点:
    > 若a∣b,b∣c,则a∣c  
    > 若a∣b,a∣c,则a∣b+c  
    > 若a∣b,对于任意的k∈ℤ,都有a∣kb  
## 同余  
**给定整数\(a\)，\(b\)和模数\(p\)，若\(a \bmod p = b \bmod p\)，则称\(a\)和\(b\)在模\(p\)意义下同余，简写成\(a\equiv b \pmod p\)**
